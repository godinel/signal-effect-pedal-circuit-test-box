# signal-effect-pedal-circuit-test-box
Box that switches between clean signal from guitar and input/output probes that can be sent through a breadboarded circuit. This allows easy testing and allows the user to switch between clean vs altered sound.

# Switch Wiring
The box uses a 3PDT latching switch and 2 1/4 inch auido jacks wired as such:

The led was omitted and the board in and out were connected to breadboard probes
